## 1. 1차원 배열
### 1.1 set 함수를 이용한 중복제거

>#### Q_3052.py   
>수 10개를 입력받은 뒤, 이를 42로 나눈 나머지를 구한다.   
그 다음 서로 다른 값이 몇 개 있는지 출력하는 프로그램을 작성하시오.
<pre>
<code>
arr=[]
for a in range(10):
    arr.append(int(input())%42)

print(len(set(arr)))
</code>
</pre>

set() 함수로 배열 변환하여 집합 자료형으로 만들어준다.   
집합자료형은 중복제거 되어 생성되기 때문에 "중복제거를 위한 필터링" 으로 이용
